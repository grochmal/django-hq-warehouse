HQ Warehouse

Available commands

The commands checkout data from the staging area and insert it into the warehouse. 

hqw-checkout-batch [-hv] -b <batch number>

  -h  Print usage.
  -v  Be verbose, print successes as well as errors.
  -b  The batch number in the staging area to attempt to fetch for the
      warehouse, it must be a valid batch number (must exist)

hqs-checkout-table [-hv] -t <table>

  -h  Print usage.
  -v  Be verbose, print successes as well as errors.
  -t  A table that may contain error rows in the staging area, it uses the
      warehouse naming convention: either `currency`, `forex` or `offer`.
Available Tables
  • currency
  • forex
  • offer (both valid and invalid are processed)

Preferably use batch numbers, since the loading order is always the correct one from a batch. If you checkout data table by table do it for the currency, forex and offer tables, in that order.
