HQ Warehouse

Available commands

The commands checkout data from the staging area and insert it into the warehouse. 

hqw-checkout-batch [-hv] -b <batch number>

hqs-checkout-table [-hv] -t <table>
Available Tables
  • currency
  • forex
  • offer (both valid and invalid are processed)

Preferably use batch numbers, since the loading order is always the correct one from a batch. If you checkout data table by table do it for the currency, forex and offer tables, in that order.
